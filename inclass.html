What does <dl> stand for?
<dl> stands for Description List element. Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs). The element encloses a list of groups of terms (defined by the <dt> element) and descriptions (marked by the <dd> elements). A single <dt> element can be followed by several <dd> elements and vice versa.
What are some different options for using tags to italicize text in HTML5?
<i> element can be used to represent text that is set off from the normal prose, such a foreign word, fictional character thoughts, or when the text refers to the definition of a word instead of representing its semantic meaning.
<cite> element should be used when citing the title of a work, such as a book or a movie.
<em> element is for words that have a stressed emphasis compared to surrounding text, which is often limited to a word or words of a sentence and affects the meaning of the sentence itself.
